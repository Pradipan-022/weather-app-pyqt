WEATHER APP (PyQt5)

  A simple and clean PyQt5-based weather application that fetches real-time weather data using the OpenWeather API.
  Users can enter any city name to view the current temperature and weather conditions.

FEATURES

  -Built with PyQt5 for a modern desktop interface

  -Fetches real-time weather data using the OpenWeather API

  -Displays temperature in Celsius

  -Maps weather condition codes to descriptive labels

  -Includes comprehensive error handling for API, network, and input issues

  -Styled UI with a responsive layout

TECHNOLODIES USED

  -Python

  -PyQt5

  -Requests

  -OpenWeather API

INSTALLATION

1. Clone the repository
  git clone https://github.com/your-username/weather-app-pyqt.git
  cd weather-app-pyqt

2. Install dependencies
  -pip install PyQt5 requests

3. Add your OpenWeather API key

  -Replace the api_key variable inside the script:
    api_key = "YOUR_API_KEY_HERE"

  -You can obtain a free API key from:
    https://openweathermap.org/api

USAGE

  -Run the application using:
    python weather_app.py

Steps:

  -Enter the name of a city

  -Click "Get Weather"

  -View the current temperature and weather description
